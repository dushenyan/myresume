# 冷邮件模板库

## 📧 核心原则

### DO ✅
- 简洁有力（150-200字）
- 突出相关性和价值
- 展示对公司的了解
- 提供具体成果数据
- 自然表达兴趣

### DON'T ❌
- 过于冗长（>300字）
- 显得急切或乞求
- 空洞的自我吹嘘
- 复制粘贴痕迹明显
- 过度使用感叹号

## 📝 定制化指南

### 如何快速定制邮件

**Step 1: 选择模板**
- 政务方向 → 模板1
- SaaS方向 → 模板2
- 可视化方向 → 模板3
- 不确定 → 模板4

**Step 2: 替换占位符**
```
[HR姓名] → 从招聘信息中找到HR名字
[公司名称] → 目标公司全称
[产品/业务方向] → 研究公司官网后填写
[手机号] → 你的手机号
[邮箱] → 你的邮箱
```

**Step 3: 个性化调整（5分钟）**
- 研究公司官网，找到1-2个具体产品或技术方向
- 在邮件中提到这些具体内容
- 说明为什么对这个方向感兴趣

**Step 4: 检查清单**
- [ ] 字数控制在150-200字
- [ ] 包含至少1个具体数据
- [ ] 提到公司具体业务
- [ ] 语气专业不急切
- [ ] 无错别字和语法错误

## 💡 高级技巧

### 1. 研究公司的3个维度
- **产品维度**：主要产品、技术栈、用户规模
- **技术维度**：技术博客、开源项目、技术分享
- **文化维度**：公司价值观、团队氛围、发展阶段

### 2. 找到共鸣点
```
❌ "我对贵司很感兴趣"
✅ "我看到贵司在微前端架构方面的探索，这与我的实践经验很契合"

❌ "希望能加入贵司"
✅ "我认同贵司在政务数字化领域的深耕，希望能参与其中"
```

### 3. 展示价值而非需求
```
❌ "我希望能在贵司学习成长"
✅ "我在性能优化方面的经验，相信能帮助团队提升产品体验"

❌ "贵司平台很好，我想加入"
✅ "我主导的项目获得省级奖项，希望能为贵司带来类似价值"
```

### 4. 时间节奏
- **首次联系**：投递简历后立即发送
- **第一次跟进**：3天后（如无回复）
- **第二次跟进**：7天后（如仍无回复）
- **放弃时机**：14天后无回复，转向其他机会

## 📊 效果评估

### 好的信号 ✅
- HR在1小时内回复
- HR询问具体技术细节
- HR主动约面试时间
- HR转发给技术负责人

### 不好的信号 ❌
- 3天以上无回复
- 回复模板化（"简历已收到，等待通知"）
- 直接拒绝且无具体原因
- 要求降低薪资预期

### 优化方向
- 回复率<10% → 优化邮件标题和开头
- 回复率10-20% → 增加个性化内容
- 回复率>20% → 保持当前策略

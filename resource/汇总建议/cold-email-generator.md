## 📝 快速定制流程（5分钟）

### Step 1: 研究公司（2分钟）

访问公司官网，找到：

- [ ] 主要产品/服务
- [ ] 技术栈（从招聘JD中找）
- [ ] 公司规模和发展阶段
- [ ] 最近的新闻或融资信息

### Step 2: 选择模板（30秒）

- 政务/国企 → 邮件1
- SaaS/创业 → 邮件3
- 在线平台 → 邮件2

### Step 3: 个性化（2分钟）

替换以下内容：

```
[HR姓名] → 从招聘信息找到（如果没有，用"招聘负责人"）
[具体业务方向] → 公司的主要产品或技术方向
[你的手机号] → 你的真实手机号
```

添加1-2句个性化内容：

```
"我注意到贵司在[X方向]的探索..."
"我看到贵司的[产品名称]很有创新性..."
"我对贵司[技术博客/开源项目]中提到的[技术点]很感兴趣..."
```

### Step 4: 检查（30秒）

- [ ] 字数在150-250字之间
- [ ] 包含至少2个具体数据
- [ ] 提到公司具体业务
- [ ] 无错别字
- [ ] 语气专业不急切

## 💡 提升回复率的5个技巧

### 1. 黄金时间发送

**最佳时间**：

- 工作日 9:00-10:00（HR刚上班）
- 工作日 14:00-15:00（午休后）

### 2. 主题行优化

```
❌ "求职"
❌ "应聘前端工程师"
```

- 包含职位名称
- 突出核心优势
- 包含姓名（方便搜索）

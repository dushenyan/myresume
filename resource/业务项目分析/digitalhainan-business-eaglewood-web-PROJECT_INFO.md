# 数字海南沉香Web管理后台项目信息

## 项目概述

**项目名称**: digitalhainan-business-eaglewood-web  
**版本**: 0.0.1  
**项目类型**: PC端管理后台系统  
**主要功能**: 沉香树木管理系统后台管理端

## 开发规范

### 提交规范
项目使用Conventional Commits规范：
- `fix`: 修复bug
- `feat`: 新功能
- `feat`: 优化
- 其他类型遵循约定式提交规范

### 代码规范
- ESLint代码检查
- Git提交前自动检查
- 支持lint-staged

## 环境配置

项目支持多环境配置：
- `.env.development`: 开发环境
- `.env.test`: 测试环境
- `.env.preview`: 预发布环境
- `.env.production`: 生产环境

## 构建命令

```bash
# 开发环境
npm run serve

# 生产构建
npm run build:prod

# 预发布构建
npm run build:preview

# 测试构建
npm run build:test

# 代码检查
npm run lint

# 单元测试
npm run test:unit

# 规范化提交
npm run commit

# SVG图标优化
npm run svgo

# 预览构建结果
npm run preview
```

## 项目特点

1. **PC端管理后台**: 使用Element UI组件库，专为PC端设计
2. **国际化**: 支持中英文切换
3. **数据可视化**: 使用ECharts进行图表展示
4. **富文本编辑**: 集成WangEditor富文本编辑器
5. **Excel处理**: 支持Excel导入导出
6. **二维码生成**: 支持生成树木二维码
7. **图片压缩**: 上传图片自动压缩
8. **权限管理**: 基于角色的权限控制
9. **路由权限**: 动态路由权限控制
10. **文件上传**: 支持多种文件上传
11. **数据加密**: 使用Crypto-js进行数据加密
12. **钉钉集成**: 支持钉钉JSAPI
13. **统一认证**: 集成UAA统一认证

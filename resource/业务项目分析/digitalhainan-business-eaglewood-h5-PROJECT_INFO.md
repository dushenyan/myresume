# 数字海南沉香H5项目信息

## 项目概述

**项目名称**: digitalhainan-business-eaglewood-h5  
**版本**: 0.0.1  
**项目类型**: 移动端H5应用  
**主要功能**: 沉香树木管理系统

## 技术栈

### 核心框架
- **Vue.js**: 2.6.11
- **Vue Router**: 3.1.5
- **Vuex**: 3.1.2

### UI组件库
- **Vant UI**: 2.10.13（移动端UI组件库）
- **Better Scroll**: 2.0.0-beta.6（滚动优化）

### 工具库
- **Axios**: 0.19.2（HTTP请求）
- **Day.js**: 1.8.23（日期处理）
- **Vee-Validate**: 3.2.5（表单验证）
- **Compressor.js**: 1.2.1（图片压缩）
- **VConsole**: 3.15.1（移动端调试工具）

### 其他依赖
- **@szhn/uaa-sdk-bridge**: 1.1.14（统一认证）
- **gdt-jsapi**: 1.9.47（钉钉集成）
- **amfe-flexible**: 2.2.1（移动端适配）

### 构建工具
- **Vue CLI**: 4.2.0
- **Babel**: 支持ES6+语法
- **Webpack**: 通过Vue CLI集成
- **PostCSS**: px转rem、px转viewport

### 代码质量工具
- **ESLint**: 6.7.2（代码检查）
- **Prettier**: 代码格式化
- **Husky**: 7.0.2（Git钩子）
- **Commitlint**: 提交信息规范
- **Lint-staged**: 暂存文件检查

## 项目结构

```
src/
├── api/              # API接口定义
├── assets/           # 静态资源
│   ├── icons/        # SVG图标
│   ├── images/       # 图片资源
│   ├── js/           # JS工具
│   ├── json/         # JSON数据
│   └── styles/       # 样式文件
├── components/       # 公共组件
├── config/           # 配置文件
├── locales/          # 国际化语言包
├── plugins/          # 插件配置
├── router/           # 路由配置
├── store/            # Vuex状态管理
├── utils/            # 工具函数
└── views/            # 页面组件
```

## 开发规范

### 提交规范
项目使用Conventional Commits规范：
- `fix`: 修复bug
- `feat`: 新功能
- `feat`: 优化
- 其他类型遵循约定式提交规范

### 代码规范
- ESLint + Prettier
- Git提交前自动检查
- 支持lint-staged

## 环境配置

项目支持多环境配置：
- `.env.development`: 开发环境
- `.env.test`: 测试环境
- `.env.preview`: 预发布环境
- `.env.production`: 生产环境

## 构建命令

```bash
# 开发环境
npm run serve

# 生产构建
npm run build:prod

# 预发布构建
npm run build:preview

# 测试构建
npm run build:test

# 代码检查
npm run lint

# 规范化提交
npm run commit
```

## 项目特点

1. **移动端优先**: 使用Vant UI组件库，专为移动端设计
2. **国际化**: 支持中英文切换
3. **性能优化**: 图片压缩、页面缓存、懒加载
4. **用户体验**: 流畅的页面切换、友好的交互反馈
5. **代码规范**: 完善的代码检查和提交规范
6. **权限管理**: 基于角色的权限控制
7. **钉钉集成**: 支持钉钉JSAPI

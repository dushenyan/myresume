# 营商环境大数据综合信息平台 - 项目分析报告

## 一、项目基本信息

**项目名称**: large-data-vision-web  
**项目描述**: 营商环境大数据综合信息平台  
**技术栈**: Vue 2.6 + Element UI + ECharts + Three.js  
**总提交次数**: 1,263次  
**开发周期**: 2023年2月 - 2023年12月  
**主要贡献者**: lichang, zhangzefei, dushenyan, 十七

## 二、项目核心功能模块

### 2.1 晾晒台系统
- **功能亮点**: 营商环境大屏晾晒台，支持大屏和小屏自适应显示
- **特色功能**: 近七日数据筛选、实时监控导出、活跃单位统计
- **优化记录**: 宽度适配、文案调整、功能优化、图标补充

### 2.2 知识库管理系统
- **核心组件**: 点子库、项目库、案例库、问题库
- **功能特性**: 
  - 支持检索功能和查询参数
  - 首页视频展示
  - 思维导图关联
  - 批注功能重新构建
  - 流程图片展示

### 2.3 12345工单系统
- **功能模块**: 工单处理流程、实时监控、详情展示
- **特色功能**: 
  - 工单步骤条展示
  - 办结状态字段
  - 查询条件优化
  - 数据验证调整

### 2.4 四大产业分析
- **产业类型**: 支持四大产业分类统计
- **数据展示**: 饼图展示、图表标识优化
- **交互功能**: 当前选中类型切换、静态图片上传

### 2.5 商事主体管理
- **核心功能**: 商事主体总数统计、注册趋势分析
- **数据管理**: 平台链接管理、增长量文案调整
- **关联功能**: 与四大产业数据关联

### 2.6 信用体系建设
- **信用指标**: 排名变动、时间更新
- **信用加模块**: 优秀案例展示、联合奖惩链接
- **数据展示**: 信用图表、案例库管理

### 2.7 政务服务监控
- **监控类型**: 政务服务办件、工规许可审批
- **功能特性**: 
  - 实时健康监控
  - 流程办理人联系方式
  - 办结状态字段
  - 数据导出功能

### 2.8 系统监控与日志
- **登录日志**: 用户行为追踪
- **系统监控**: 业务系统监控、活跃度统计
- **数据统计**: 月均浏览量、单位活跃度

## 三、技术亮点与创新

### 3.1 大屏可视化技术
- **3D地图展示**: 使用Three.js实现3D地图，支持放大缩小、点击交互
- **地图连线**: LeaderLine实现地图连线，解决放大时定位问题
- **图表组件**: 集成ECharts和ECharts GL，支持多种图表类型
- **翻牌器动画**: FlipNumber组件实现数字动态翻转效果

### 3.2 响应式适配方案
- **vw适配方案**: 使用postcss-px-to-viewport实现大屏适配
- **双模式支持**: 支持px开发和vw生产环境转换
- **屏幕适配**: 弹窗适配屏幕宽度、架构图适配小屏幕
- **设备兼容**: 海政通环境下文字裁剪修复

### 3.3 组件化架构
- **自定义组件库**: data-vision-ui大屏UI组件库
- **组件分类**: 
  - 通用组件（PascalCase命名）
  - 业务组件（kebab-case命名）
- **核心组件**: Box、Chart、Image、Wrapper、TopHeader等

### 3.4 数据安全与权限
- **数据脱敏**: 手机号和座机号脱敏处理
- **角色权限**: 用户角色权限判断、脱敏角色判断
- **访问控制**: 大屏人员允许访问简略版

### 3.5 交互体验优化
- **批注功能**: 重新构建批注系统
- **思维导图**: 问题思维导图、连线对齐修复
- **实时更新**: 实时监控流程显示优化
- **动画效果**: 翻牌器、进度条、数字增长动画

## 四、被处理和优化的问题

### 4.1 样式与布局问题
- **表格问题**: 修复表格高度、浏览量换行、右侧菜单间隔不一致
- **图表问题**: 修复图表放大右边遮挡、tooltip图片变形、图表单位添加
- **弹窗问题**: 调整弹框宽度、弹窗适配屏幕宽度
- **文字问题**: 修复文字截断、文字被裁、标题不一致

### 4.2 数据处理问题
- **数据兼容**: 十八项指标兼容非数字值、处理每个年份的数据
- **数据验证**: 工单数据显示验证调整、表格回显数据判断
- **数据展示**: 系统浏览量改为月均浏览量、默认展示活跃用户
- **数据导出**: 实时健康导出不要分页、实时监控导出

### 4.3 接口与联调问题
- **接口联调**: 业务系统一级页面接口联调完成、营商大屏接口联调
- **链接管理**: 商事主体平台链接修改、12345大屏地址调整
- **数据写死**: 政务服务数据写死、重大任务督办写死
- **假数据清理**: 去掉业务系统表格假数据

### 4.4 功能逻辑问题
- **查询优化**: 实时监控列表添加查询参数、问题库参数修复
- **分页问题**: 重置分页问题修复
- **排序问题**: 单位活跃度和活跃用户调换顺序、修改同比的排版
- **状态管理**: 指标问题字段调整、办结状态字段

### 4.5 性能与兼容性
- **地图优化**: 禁用地图click-tooltip、修改地图LeaderLine的class和id避免冲突
- **组件优化**: 切分组件、优化样式、组件命名规范化
- **代码优化**: 删除indexjs文件、删除无用样式、remove console
- **环境适配**: 火熊修改为信创环境、接入土熊SDK

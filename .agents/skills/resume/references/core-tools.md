# 核心求职诊断工具

本技能提供7个AI驱动的求职诊断工具，帮助用户从简历优化到面试通关，提高求职成功率。

## 1. 零回复诊断

以资深招聘官视角深度分析：

- 简历定位
- 关键词匹配
- 经验表述
- 与目标职位的契合度

找出投递无果的根本原因。

## 2. 目标职位锚定

基于用户的背景与技能，精准锁定最匹配的职位方向。剔除虚泛选项，确立聚焦的求职赛道。

**使用方法**：

1. 用户提供背景信息
2. 技能分析用户背景与技能
3. 输出最匹配的职位方向

## 3. 应聘策略系统

制定结构化求职方案：

- 投递数量规划
- 简历定制方法
- 进度追踪机制

打造可落地的执行系统。

## 4. 招聘官视角重写

模拟10秒简历扫描场景：

- 识别亮点与盲区
- 重写关键模块
- 提升信息密度与冲击力

## 5. 冷邮件生成器

针对目标职位撰写专业 outreach 信息：

- 精准传递价值
- 表达兴趣
- 自然不刻意

**使用方法**：

1. 用户提供职位名称
2. 技能生成专业的冷邮件

## 6. 面试模拟器

扮演目标职位招聘经理，进行全真模拟面试：

- 针对性提问
- 即时反馈
- 打磨表达清晰度与临场自信

**使用方法**：

1. 用户提供目标职位
2. 技能扮演招聘经理进行模拟面试

## 7. 求职全案审计

系统性复盘：

- 简历
- 定位
- 外联
- 面试准备

输出可量化的优化方案，最大化面试转化率。

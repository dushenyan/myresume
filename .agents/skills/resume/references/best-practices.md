# 简历优化与求职最佳实践

## 使用方法

1. 用户选择需要的工具或提示词模板
2. 提供必要的信息（如背景信息、职位名称等）
3. 技能生成相应的分析、建议或内容
4. 用户根据输出进行优化和调整

## 注意事项

- 提供的信息越详细，技能的输出越准确
- 对于需要提供特定信息的工具，请确保信息完整
- 技能的输出是基于AI分析，建议结合个人实际情况进行调整
- 定期更新简历和求职策略，以适应市场变化

## 常见问题

### Q: 如何选择合适的工具？

A: 根据你的具体需求选择：

- 投递无果：使用零回复诊断
- 职业方向不明确：使用目标职位锚定
- 需要系统的求职计划：使用应聘策略系统
- 简历需要优化：使用招聘官视角重写
- 需要联系招聘方：使用冷邮件生成器
- 面试准备：使用面试模拟器
- 全面优化：使用求职全案审计

### Q: 提示词模板如何使用？

A: 选择适合你情况的提示词模板，填入相应的信息，然后使用生成的内容创建或优化你的简历。

### Q: 技能的分析结果有多准确？

A: 技能基于资深招聘官的视角和AI分析，提供专业的建议和分析，但最终决策还需结合个人实际情况。

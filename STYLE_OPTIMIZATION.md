# 样式优化说明

## 🎨 优化内容

### 1. 响应式布局优化

#### 桌面端 (≥1200px)
- 字体大小：16px 基础字号，姓名 3.2rem
- 网格布局：左侧个人信息 + 右侧内容区域
- 技能标签：网格布局，每行自适应显示
- 卡片间距：2.5rem 内边距，15px 外边距

#### 平板端 (768px-1199px)
- 字体大小：15px 基础字号，姓名 2.5rem
- 网格布局：2列布局，优化空间利用
- 技能标签：缩小尺寸，保持可读性
- 头像尺寸：90x90px

#### 手机端 (<768px)
- 字体大小：14px 基础字号，姓名 2.2rem
- 单列布局：垂直堆叠，优化触摸体验
- 技能标签：2列布局，紧凑显示
- 头像尺寸：80x80px

### 2. 打印优化

#### 页面设置
- 纸张：A4 (210×297mm)
- 边距：1.5cm
- 字体：10pt 基础字号

#### 内容优化
**显示内容：**
- 个人基本信息（姓名、职位、邮箱、电话）
- 自我描述
- 项目经历（简化版）
- 工作经验
- 专业技能
- 自我评价
- 教育经历

**隐藏内容：**
- 社交链接（GitHub、Twitter等）
- 项目详细描述
- 项目成就列表
- 专业认证
- 技术出版物
- 荣誉奖项
- 兴趣领域
- 底部PDF链接和更新时间

#### 打印样式特性
- 防止重要内容分页断开
- 优化间距，节省纸张
- 黑白优化，确保可读性
- 移除装饰性元素

### 3. 视觉改进

#### 技能标签
- 渐变背景效果
- 悬停动画（上移+光效）
- 网格布局自适应
- 响应式尺寸调整

#### 卡片设计
- 圆角边框
- 阴影效果
- 悬停交互反馈
- 统一间距规范

#### 自我评价
- 图标标记（✓）
- 左边框分隔线
- 优化行间距
- 防止分页断开

## 🔧 技术实现

### 样式架构
```
assets/less/
├── theme.less          # 主样式文件
├── base.less           # 基础和响应式样式
├── print.less          # 打印专用样式
├── cards.less          # 卡片组件样式
└── sections/
    ├── skills.less     # 原始技能样式
    ├── skills-custom.less # 自定义技能样式
    └── references.less # 引用样式
```

### 关键改进
1. **CSS Grid 布局**：响应式网格系统
2. **Flexbox 布局**：灵活的元素排列
3. **CSS 变量**：统一的设计令牌
4. **媒体查询**：精细的响应式断点
5. **打印样式**：专门的@media print规则

## 🎯 使用方法

### 开发环境
```bash
# 编译开发样式
npm run compile

# 构建完整页面
npm run build:html

# 启动开发服务器
npm start
```

### 样式预览
- 桌面端：推荐1920×1080分辨率
- 平板端：推荐768px宽度
- 手机端：推荐375px宽度
- 打印预览：使用浏览器打印功能

## 📋 注意事项

1. **字体加载**：使用Google Fonts，确保网络连接
2. **图片优化**：头像建议使用正方形图片
3. **打印设置**：建议使用Chrome的"保存为PDF"功能
4. **响应式测试**：使用开发者工具模拟不同设备

## 🔄 热更新集成

样式修改后，由于之前配置的热更新功能，系统会自动：
1. 检测resumeSource.ts变化
2. 重新生成resume.json
3. 重新编译HTML
4. 开发服务器自动刷新

确保开发时的实时预览体验。

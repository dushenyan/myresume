## 简历分析

1. 零回复诊断 '以资深招聘官身份，分析我的简历、目标职位和申请历史。找出我未获面试的最可能原因。突出定位问题、关键词缺陷、经验表述弱点或职位不匹配。’
2. 目标职位明确化 '根据我的背景和技能，确定最现实的职位名称。排除不匹配或过于宽泛的角色。制定聚焦的求职方向。背景信息：[粘贴]。'
3. 应聘策略构建器 '基于目标职位和经验，创建结构化的求职策略。定义应聘数量、定制方法和进度跟踪方式。’
4. 招聘官视角重写 '从招聘官10秒扫描视角审查我的简历和[招聘平台]。指出什么突出、什么不清楚。重写关键部分以提高清晰度和影响力。’
5. 冷邮件生成器 '为职位[职位名称]起草简洁专业的信息。展现相关性、价值和兴趣，避免显得急切。'
6. 面试问题模拟器 '扮演[职位]的招聘经理。进行真实模拟面试。提问相关问题并评估回答。给出关于清晰度、结构和信心的反馈。'
7. 最终求职审计 '审查我的完整求职方法，包括简历、定位策略、外联和面试准备。优化成提高面试概率的结构化系统。’
